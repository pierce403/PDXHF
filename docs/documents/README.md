# Documents directory

This directory is for the core organizational and transparency documents that should eventually be linked from the public site.

Suggested files to add here:

## Core nonprofit documents

- `irs-determination-letter.pdf`
  - IRS recognition of 501(c)(3) status
- `form-1023.pdf` or `form-1023-ez.pdf`
  - exemption application, if you want to publish it
- `ein-letter.pdf`
  - EIN confirmation letter, if useful

## Annual filings

Add the most recent filings first, then backfill older ones as needed.

Suggested naming:

- `form-990-2025.pdf`
- `form-990-ez-2025.pdf`
- `form-990-n-2025.pdf`

Only one of those would typically apply for a given year.

## Governance documents

- `bylaws.pdf`
- `conflict-of-interest-policy.pdf`
- `board-roster-2026.md` or `.pdf`
- `grant-process.md` or `.pdf`

## Optional transparency documents

These are not always legally required to publish on the site, but they can be useful trust signals.

- `annual-report-2025.pdf`
- `board-minutes-2026-01-15.pdf`
- `board-minutes-2026-02-20.pdf`
- `grant-awards-2026.md`
- `donor-transparency-policy.md`

## Notes

- PDFs are good for signed/final documents.
- Markdown is good for living documents such as board rosters, grant process notes, or grant award summaries.
- Once documents are uploaded here, we can add cleaner links from `transparency.md` and the homepage/footer.
